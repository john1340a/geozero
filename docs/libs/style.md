# HeroUI & Tailwind CSS

## HeroUI (anciennement NextUI)

**HeroUI** est une bibliothèque de composants UI moderne, rapide et belle, construite sur React et Tailwind CSS.

- **Documentation Officielle** : [https://www.heroui.com/](https://www.heroui.com/)
- **Utilité dans le projet** : (Prévu initialement) Fournit des composants stylisés et accessibles. _Note : Le projet utilise actuellement beaucoup de CSS personnalisé (Glassmorphism) et index.css._

## Tailwind CSS

**Tailwind CSS** est un framework CSS "utility-first" qui permet de construire des designs rapidement sans sortir du HTML.

- **Documentation Officielle** : [https://tailwindcss.com/](https://tailwindcss.com/)
- **Utilité dans le projet** : Utilisé pour la configuration de base et certains utilitaires de style, bien que le projet ait évolué vers des styles CSS personnalisés pour l'effet "Glass".
