# Fast XML Parser

**Fast XML Parser** est une bibliothèque ultra-rapide pour valider et parser des données XML en JSON (et inversement).

- **Documentation Officielle** : [https://github.com/NaturalIntelligence/fast-xml-parser](https://github.com/NaturalIntelligence/fast-xml-parser)
- **Utilité dans le projet** : Indispensable pour transformer le flux RSS de GeoRezo (qui est en XML) en objets JavaScript utilisables par l'application pour afficher les offres d'emploi.
