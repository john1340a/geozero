# Playwright

**Playwright** est un framework permettant d'effectuer des tests de bout en bout (E2E) fiables pour les applications web modernes.

- **Documentation Officielle** : [https://playwright.dev/](https://playwright.dev/)
- **Utilité dans le projet** : Assure que l'application fonctionne correctement du point de vue de l'utilisateur.
  - Teste le chargement de la page.
  - Vérifie la présence de la carte et des marqueurs.
  - Simule les interactions utilisateur.

### Commande pour lancer les tests

```bash
npx playwright test
```
