# Leaflet & React-Leaflet

## Leaflet

**Leaflet** est la bibliothèque JavaScript open-source de référence pour les cartes interactives adaptées aux mobiles. Elle est légère, performante et facile à utiliser.

- **Documentation Officielle** : [https://leafletjs.com/](https://leafletjs.com/)
- **Utilité dans le projet** : Moteur de rendu de la carte principale, gestion des tuiles (OpenStreetMap) et des marqueurs.

## React-Leaflet

**React-Leaflet** fournit des liaisons (bindings) React pour Leaflet. Elle permet d'utiliser les composants Leaflet (MapContainer, TileLayer, Marker, Popup) comme des composants React déclaratifs.

- **Documentation Officielle** : [https://react-leaflet.js.org/](https://react-leaflet.js.org/)
- **Utilité dans le projet** : Intégration de Leaflet dans l'architecture React de l'application.
